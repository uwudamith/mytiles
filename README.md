This is Spring MVC project and following technologies are associated with the project
<ul>
  <li>Spring MVC</li>
  <li>Spring Security</li>
  <li>Maven</li>
  <li>MySQL</li>
  <li>jetty</li>

  <li>JSP</li>

</ul>
